📊 Exercícios de Banco de Dados SQL

Este repositório contém exercícios práticos de modelagem e manipulação de banco de dados utilizando SQL.
Os projetos foram desenvolvidos com foco em aprendizado dos conceitos fundamentais utilizados no desenvolvimento back-end e análise de dados.

🧠 Conceitos praticados
Criação de banco de dados (CREATE DATABASE)
Criação de tabelas (CREATE TABLE)
Tipos de dados (VARCHAR, INT, DECIMAL, DATE)
Inserção de dados (INSERT)
Consultas (SELECT)
Filtros com WHERE
Atualização de dados (UPDATE)
Chaves primárias (PRIMARY KEY)
(Em alguns casos) relacionamento com chave estrangeira (FOREIGN KEY)

📁 Estrutura do repositório
banco-de-dados/
├── rh.sql
├── ecommerce.sql

👥 Projeto 1: RH (Recursos Humanos)
Sistema simples para gerenciamento de colaboradores.

📌 Funcionalidades:
Cadastro de colaboradores
Consulta por faixa salarial
Atualização de salário
📊 Atributos utilizados:
id
nome
cargo
salario
data_admissao

🛒 Projeto 2: E-commerce
Simulação de um sistema de produtos de uma loja online.

📌 Funcionalidades:
Cadastro de produtos
Controle de estoque
Consulta por faixa de preço
Atualização de valores
📊 Atributos utilizados:
id
nome
descricao
preco
estoque

🎓 Projeto 3: Escola
Sistema básico de registro de estudantes.

📌 Funcionalidades:
Cadastro de estudantes
Consulta por desempenho (notas)
Atualização de notas
📊 Atributos utilizados:
id
nome
idade
turma
nota
cpf

🚀 Objetivo
Este repositório tem como objetivo consolidar conhecimentos em banco de dados, servindo como base para projetos mais avançados e aplicações reais.
